# Julklappslotteri

Den här appen hjälper en grupp släktingar eller vänner att på ett rättvist och transparent sätt lotta vem som ska köpa julklappar till vem. Allt sker direkt i webbläsaren, utan inloggning eller lagring på server, vilket gör den enkel att använda tillsammans när ni planerar julklappsutbytet.

Funktioner:
* Ange en rubrik för lottningen (t.ex. “Julklappslotteri 2025 – Släkten Persson”).
* Lägg in alla deltagare genom att skriva ett namn per rad.
* Bestäm hur många julklappar varje person ska köpa.
* Ange ett riktvärde för det totala värdet på julklappar per person (t.ex. 300 kr).
* Lägg till “redan köpta klappar” (vem som redan köpt till vem) så att befintliga köp respekteras i lottningen.

Algoritmen säkerställer att:
* Ingen köper till sig själv.
* Alla köper lika många julklappar som de får.
* De “låsta” paren (redan köpta klappar) inkluderas.

Resultatet kan:
* Laddas ner som en bild (PNG) med rubrik, antal julklappar och värde per person.
* Kopieras som text för att klistras in i t.ex. mejl, chatt eller dokument.
